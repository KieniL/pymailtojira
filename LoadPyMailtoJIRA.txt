Python und Git installieren

pip install virtualenv


Virtualenvironment erstellen:
virtualenv <virtualenvname>

git remote add pg git://github.com/Kreidl/pymailtojira.git


git checkout pg Mappingtable.xlsx

git checkout main.py

in Directory: Scripts\activate um Virtualenvironment aufzurufen
